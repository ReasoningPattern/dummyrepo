

To implement the `popitem` method, we'll assume the class uses a dictionary to store key-value pairs and follows the behavior of a least recently used (LRU) cache. Each access moves the item to the end, making it the most recently used. The `popitem` method removes and returns the least recently used item.

```python
def popitem(self):
    """
    Remove and return the (key, value) pair least recently used.
    """
    key, value = self.data.popitem()
    return (key, value)
```
